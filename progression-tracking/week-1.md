# Project 1: Create a program that calculates the distance between two points in 2D space.

# Step 1.1: Define variables to store the coordinates of the two points.
# Step 1.2: Prompt the user to input the coordinates of the two points.
# Step 1.3: Convert the user input into numerical values and assign them to the variables.
# Step 1.4: Calculate the distance between the two points using the distance formula.
# Step 1.5: Output the result to the user.

# Project 2: Create a program that calculates the area of a circle given its radius.

# Step 2.1: Define a variable to store the radius of the circle.
# Step 2.2: Prompt the user to input the radius of the circle.
# Step 2.3: Convert the user input into a numerical value and assign it to the variable.
# Step 2.4: Calculate the area of the circle using the formula for area.
# Step 2.5: Output the result to the user.

# Project 3: Develop a program that converts a temperature given in Celsius to Fahrenheit or vice versa.

# Step 3.1: Define variables to store the temperature in Celsius and Fahrenheit.
# Step 3.2: Prompt the user to input a temperature and its unit (Celsius or Fahrenheit).
# Step 3.3: Convert the user input into numerical values and assign them to the appropriate variables.
# Step 3.4: Calculate the temperature in the other unit using the conversion formulas.
# Step 3.5: Output both temperatures to the user.

# Project 4: Create a program that takes in a sentence and outputs the number of vowels in the sentence.

# Step 4.1: Define a variable to store the number of vowels.
# Step 4.2: Prompt the user to input a sentence.
# Step 4.3: Loop through each character in the sentence and check if it is a vowel.
# Step 4.4: If the character is a vowel, increment the vowel count.
# Step 4.5: Output the vowel count to the user.

# Project 5: Build a program that calculates the factorial of a given number.

# Step 5.1: Define variables to store the input number and the factorial result.
# Step 5.2: Prompt the user to input a number.
# Step 5.3: Convert the user input into a numerical value and assign it to the variable.
# Step 5.4: Calculate the factorial of the input number using a loop.
# Step 5.5: Output the factorial result to the user.

# Project 6: Build a program that simulates rolling a pair of dice and outputs the sum of the dice rolls.

# Step 6.1: Define variables to store the results of two dice rolls and their sum.
# Step 6.2: Generate random numbers between 1 and 6 for each die roll.
# Step 6.3: Add up the results of both dice rolls.
# Step 6.4: Output the sum of the dice rolls to the user.

# Project 7: Develop a program that takes in a string and outputs the string in reverse.

# Step 7.1: Define variables to store the original string and its reversed version.
# Step 7.2: Prompt the user to input a string.
# Step 7.3: Loop through each character in the string in reverse order and append it to the reversed version variable.
# Step 7.4: Output the reversed string to the user.

# Project 8: Create a program that checks if a given word is a palindrome (i.e., reads the same backward as forward).

# Step 8.1: Define variables to store the original word and its reversed version.
# Step 8.2: Prompt the user to input a word.
# Step 8.3: Loop through each character in the word in reverse order and append it to the reversed version variable.
# Step 8.4: Check if the original word and its reversed version are equal.
# Step 8.5: Output whether or not the word is a palindrome to the user.

# Project 9: Develop a program that calculates the roots of a quadratic equation.

# Step 9.1: Define variables to store the coefficients of the quadratic equation.
# Step 9.2: Prompt the user to input the coefficients.
# Step 9.3: Convert the user input into numerical values and assign them to the appropriate variables.
# Step 9.4: Calculate the roots of the quadratic equation using the quadratic formula.
# Step 9.5: Output the roots to the user.

# Project 10: Develop a program that generates a Fibonacci sequence up to a given number.

# Step 10.1: Define variables to store the current and previous numbers in the sequence, as well as a limit value.
# Step 10.2: Prompt the user to input a limit value.
# Step 10.3: Convert the user input into a numerical value and assign it to the variable.
# Step 10.4: Generate each number in the Fibonacci sequence up to the limit value using a loop.
# Step 10.5: Output each number in the sequence to the user.

# Project 11: Create a program that takes in a list of numbers and outputs the largest and smallest numbers in the list.

# Step 11.1: Define variables to store the largest and smallest numbers in the list.
# Step 11.2: Prompt the user to input a list of numbers.
# Step 11.3: Convert the user input into a list of numerical values and assign it to a variable.
# Step 11.4: Loop through each number in the list and compare it to the current largest and smallest values.
# Step 11.5: Output the largest and smallest numbers to the user.

# Project 12: Create a program that allows the user to input their age and outputs the number of days, hours, and minutes they have been alive.

# Step 12.1: Define variables to store the user's age in years, days, hours, and minutes.
# Step 12.2: Prompt the user to input their age in years.
# Step 12.3: Convert the user input into a numerical value and assign it to the variable.
# Step 12.4: Calculate the number of days, hours, and minutes the user has been alive using conversion formulas.
# Step 12.5: Output the results to the user.

# Project 13: Build a program that generates a random number between 1 and 100 and allows the user to guess the number. The program should give the user hints if their guess is too high or too low.

# Step 13.1: Define variables to store the randomly generated number and the user's guess.
# Step 13.2: Generate a random number between 1 and 100.
# Step 13.3: Prompt the user to input their guess.
# Step 13.4: Convert the user input into a numerical value and assign it to the variable.
# Step 13.5: Compare the user's guess to the randomly generated number and output whether it is too high or too low. Repeat steps 13.3-13.5 until the user guesses the correct number.

# Project 14: Build a program that generates a random password consisting of lowercase and uppercase letters, numbers, and symbols.

# Step 14.1: Define variables to store the characters allowed in the password and the length of the password.
# Step 14.2: Prompt the user to input the desired length of the password.
# Step 14.3: Convert the user input into a numerical value and assign it to the variable.
# Step 14.4: Generate a random password by selecting characters from the allowed characters list using a loop.
# Step 14.5: Output the generated password to the user.

# Project 15: Create a program that calculates the area of a triangle, given the length of its sides using Heron's formula.

# Step 15.1: Define variables to store the lengths of the three sides of the triangle.
# Step 15.2: Prompt the user to input the lengths of the three sides.
# Step 15.3: Convert the user input into numerical values and assign them to the appropriate variables.
# Step 15.4: Calculate the area of the triangle using Heron's formula.
# Step 15.5: Output the area of the triangle to the user.

# Project 16: Develop a program that generates a Koch curve, using recursion and iteration.

# Step 16.1: Define variables to store information about each segment of the Koch curve (e.g., start point, end point, angle).
# Step 16.2: Prompt the user to input parameters for generating the Koch curve (e.g., starting line length, number of iterations).
# Step 16.3: Convert the user input into numerical values and assign them to appropriate variables.
# Step 16.4: Use recursion and iteration to generate each segment of the Koch curve.
# Step 16.5: Output the final Koch curve to the user.

# Project 17: Create a program that sorts a list of numbers using bubble sort.

# Step 17.1: Define a list of numbers to be sorted.
# Step 17.2: Use a loop to iterate through the list.
# Step 17.3: Compare adjacent elements and swap them if they are in the wrong order.
# Step 17.4: Repeat the loop until no more swaps are needed.
# Step 17.5: Output the sorted list to the user.
